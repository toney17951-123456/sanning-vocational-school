### 情况2：链接到新页面（需要配置路由）

#### 操作步骤：
1. **创建新页面组件**：在 `src/views/` 目录下创建新的Vue文件，如 `ContactView.vue`
2. **配置路由**：在 `router/index.js` 文件中添加新路由
3. **添加导航项**：在 `App.vue` 文件中添加导航链接

#### 详细操作：

##### 步骤1：创建新页面组件
- 在 `src/views/` 目录下创建 `ContactView.vue` 文件
- 内容示例：
  ```vue
  <template>
    <div class="contact-container">
      <h2>联系我们</h2>
      <p>这里是联系我们页面的内容</p>
    </div>
  </template>
  
  <script setup>
  // 组件逻辑
  </script>
  
  <style scoped>
  .contact-container {
    padding: 2rem;
  }
  </style>
  ```

##### 步骤2：配置路由
- 打开 `router/index.js` 文件
- 在第3-40行的路由数组中添加新路由：
  ```javascript
  const routes = [
    // 现有路由...
    {
      path: '/contact',  // 路由路径
      name: 'Contact',   // 路由名称
      component: () => import('../views/ContactView.vue')  // 组件路径
    }
  ]
  ```

##### 步骤3：添加导航项
- 打开 `App.vue` 文件
- 在导航菜单中添加新的链接：
  ```html
  <nav>
    <!-- 现有导航项... -->
    <router-link to="/contact">联系我们</router-link> <!-- 新增导航项 -->
    <el-button type="primary" @click="$router.push('/apply')">在线报名</el-button>
  </nav>
  ```

## 六、注意事项

1. **路由路径要匹配**：导航链接的 `to` 属性值必须与 `router/index.js` 中配置的 `path` 一致
2. **组件路径要正确**：配置新路由时，`component` 中的文件路径要正确
3. **保存所有修改**：修改完成后，一定要保存所有修改的文件
4. **刷新页面查看效果**：修改后需要刷新浏览器页面才能看到效果
5. **语法要正确**：修改代码时要注意语法正确性，避免出现拼写错误或缺少符号等问题
