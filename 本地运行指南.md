# 三宁职校官网本地运行指南

本文档详细说明如何在本地计算机上运行三宁职校官网，无需部署到服务器。

## 一、环境准备

在开始之前，请确保您的计算机已安装以下软件：

- **Node.js** (版本 16.x 或更高)
  - 下载地址：[https://nodejs.org/](https://nodejs.org/)
  - 安装完成后，打开命令提示符(CMD)或PowerShell，运行 `node -v` 和 `npm -v` 验证安装

- **Python** (版本 3.9 或更高)
  - 下载地址：[https://www.python.org/](https://www.python.org/)
  - 安装时请勾选 "Add Python to PATH"
  - 安装完成后，运行 `python --version` 和 `pip --version` 验证安装

- **MySQL** (版本 8.0)
  - 下载地址：[https://dev.mysql.com/downloads/mysql/](https://dev.mysql.com/downloads/mysql/)
  - 安装过程中，需要设置root密码（默认为'password'）

## 二、获取项目代码

确保您已经有了完整的项目代码文件夹。如果没有，请从代码仓库获取。

项目结构如下：
```
三宁职校官网（第5稿）/
├── frontend/           # 前端项目
├── backend/            # 后端项目
├── README.md           # 项目说明
└── 其他文档文件
```

## 三、启动前端服务

### 步骤1：安装前端依赖

1. 打开命令提示符(CMD)或PowerShell
2. 进入前端项目目录：
   ```bash
   cd 三宁职校官网（第5稿）\frontend
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
   等待安装完成（可能需要几分钟）

### 步骤2：启动前端开发服务器

1. 继续在前端目录下执行：
   ```bash
   npm run dev
   ```
2. 成功启动后，终端将显示类似以下信息：
   ```
   VITE v6.x.x  ready in XXXX ms
   ➜  Local:   http://localhost:5173/
   ➜  Network: use --host to expose
   ```
3. 不要关闭此命令窗口，让服务保持运行状态

## 四、启动后端服务

### 步骤1：配置环境变量

1. 打开新的命令提示符(CMD)或PowerShell窗口
2. 进入后端项目目录：
   ```bash
   cd 三宁职校官网（第5稿）\backend
   ```
3. 检查是否存在 `.env` 文件，如果不存在，则复制 `.env.example` 并重命名为 `.env`：
   ```bash
   copy .env.example .env
   ```
4. 打开 `.env` 文件（使用记事本或其他编辑器），检查并修改以下配置：
   ```
   # 数据库配置（默认配置如下）
   DATABASE_URI=mysql+pymysql://root:password@localhost:3306/saning_school
   
   # 密钥配置（建议修改为更安全的值）
   SECRET_KEY=development-secure-secret-key-2024
   JWT_SECRET_KEY=development-jwt-secret-key-2024
   ```

### 步骤2：安装后端依赖

1. 在后端目录下执行：
   ```bash
   pip install -r requirements.txt
   ```
   等待安装完成

### 步骤3：创建数据库

1. 打开MySQL客户端（如MySQL Command Line Client或Navicat）
2. 登录MySQL（使用root用户和您设置的密码）
3. 创建数据库：
   ```sql
   CREATE DATABASE saning_school CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   CREATE USER 'saning_user'@'localhost' IDENTIFIED BY 'password';
   GRANT ALL PRIVILEGES ON saning_school.* TO 'saning_user'@'localhost';
   FLUSH PRIVILEGES;
   ```

### 步骤4：启动后端开发服务器

1. 回到后端项目目录的命令窗口
2. 执行：
   ```bash
   python app.py
   ```
3. 成功启动后，终端将显示类似以下信息：
   ```
   * Serving Flask app 'app'
   * Debug mode: on
   * Running on http://127.0.0.1:5000
   * Restarting with watchdog (windowsapi)
   * Debugger is active!
   ```
4. 不要关闭此命令窗口，让服务保持运行状态

## 五、访问网站

1. 确保前端和后端服务都已成功启动
2. 打开浏览器（推荐Chrome、Firefox或Edge）
3. 在地址栏输入：[http://localhost:5173/](http://localhost:5173/)
4. 您现在应该可以看到三宁职校官网的首页了

## 六、停止服务

当您完成使用后，可以按以下步骤停止服务：

1. 回到前端服务的命令窗口，按下 `Ctrl + C` 键
2. 回到后端服务的命令窗口，同样按下 `Ctrl + C` 键

## 七、常见问题排查

### 问题1：前端依赖安装失败

- **解决方法**：
  - 确保Node.js版本符合要求
  - 尝试使用管理员权限运行命令窗口
  - 执行 `npm cache clean --force` 后重新安装

### 问题2：MySQL连接失败

- **解决方法**：
  - 检查MySQL服务是否正在运行（可在任务管理器或服务中查看）
  - 确认.env文件中的数据库连接字符串是否正确
  - 验证root密码是否与安装时设置的一致
  - 确保已创建名为saning_school的数据库

### 问题3：端口被占用

- **症状**：启动服务时提示 "port already in use"
- **解决方法**：
  - 找到占用端口的程序并关闭它
  - 或者修改服务端口（修改配置文件）

### 问题4：页面可以打开但无法加载数据

- **解决方法**：
  - 检查后端服务是否正常运行
  - 确认前后端服务之间的API调用路径是否正确
  - 查看浏览器开发者工具的控制台错误信息

## 八、开发注意事项

1. 此运行方式仅用于开发和测试环境，生产环境需要部署到正式服务器
2. 前端开发服务器支持热重载，修改代码后页面会自动刷新
3. 后端开发服务器也开启了调试模式，代码修改后会自动重启
4. 数据库需要手动创建，系统不会自动生成数据库结构

如果您在运行过程中遇到其他问题，请参考项目的技术文档或寻求技术支持。
